# G-Scores

G-Scores is a Ruby on Rails to search 

### Requirements

- Name: g-scores
- Ruby version: 3.3.4
- Rails version: 7.1.4
- Bundler version: 2.55.2
- Database: PostgreSQL

### Installation Steps

1. Clone the repository:
```bash
git clone https://github.com/dkdeptrai/g-scores.git
```

2. Install dependencies:
```bash
bundle install
```
3. Database setup:
```bash
rails db:create db:migrate db:seed
```

4. Start the server:
```bash
rails s
```
